# AI Guide Index  開発ガイド

本ディレクトリは、 templateリポジトリでコードを扱う際の AI 支援開発用ガイドです。
AI Agent ツールと協働して、効率的かつ正確に開発を行うための包括的なドキュメントを提供します。

## 全体ルール

### 言語に関するルール

**すべてのコミュニケーションは日本語で行ってください:**

- **コミットメッセージ**: 必ず日本語で記述（例: `feat: ユーザー認証機能を追加`）
- **プルリクエスト**: タイトルと説明は日本語で記述
- **コードコメント**: 日本語を推奨
- **ドキュメント**: 日本語で記述
- **AIとのやり取り**: すべて日本語で行う

技術用語は英語のまま使用可能です（例: 「Clean Architecture」「refactor」など）。

詳細は [commit-and-pr.md](commit-and-pr.md) を参照してください。
